 To-Do List

Um projeto simples feito com Next.js e React, criado para praticar meus primeiros passos com JavaScript e JSX.  
O objetivo é aprender como o React atualiza a tela em tempo real e como organizar um projeto moderno.

 Tecnologias usadas
 
- Next.js
- React
- JavaScript
- CSS

---

 Funcionalidades
 
- Adicionar novas tarefas
- Exibir lista de tarefas
- Interface simples e centralizada

---

![Prévia do Projeto](https://github.com/dxniel-dv/to-do-list-nextjavascript/blob/main/image.png?raw=true)

Visual

O projeto tem um fundo escuro com texto branco e uma caixinha centralizada — feito para ser o mais simples possível, ideal para iniciantes.
